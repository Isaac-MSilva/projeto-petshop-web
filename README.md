# Projeto PetShop Patinhas Felizes

Este projeto foi desenvolvido para as Fases 1 e 2 da disciplina de Fundamentos de Sistemas Web.

## Funcionalidades
- **Página Inicial:** Carrossel de imagens e apresentação.
- **Produtos:** Listagem de produtos com Grid Layout.
- **Serviços:** Detalhes de banho/tosa e botão de agendamento.
- **Agendamento:** Formulário completo com validação e alerta.
- **Rodapé:** Relógio em tempo real (Função Temporal).
- **Acessibilidade:** Todas as imagens possuem descrição (alt).

## Tecnologias
- HTML5
- CSS3 (Bootstrap 5)
- JavaScript
